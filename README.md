\# SpendWise - Personal Finance Tracker



A simple, clean Finance Tracker built with \*\*Vanilla JavaScript\*\*. This project demonstrates DOM manipulation, event handling, and array methods.



\## Features

\* Add income and expenses.

\* Real-time balance calculation.

\* Transaction history list.

\* Color-coded entries (Red for expenses, Green for income).



\## Tech Stack

\* HTML5

\* CSS3

\* JavaScript (ES6+)
